vue学习(二)

2019-09-22
Axios, router, vue, vuex

Vue全家桶

  1. 在vue中使用sass的配置方法。
  • 安装sass依赖包
    npm install -save-dev sass-loader
    npm install –save-dev node-sass
  • 在build文件夹下的webpack.base.conf.js 的rules里面添加配置
    {test：/\.sass$/,loaders:['style','css','sass']}
  • 在代码中的style标签中，加上lang=”sass”属性即可~
  1. 路由

  • 在实际开发中，app.vue只是作为一个存放组件的容器，用<router-view/>来渲染你通过路由渲染过来的组件，当路径改变，<router-view>中的内容也会发生更改。

  • 当点击导航菜单时，会切换tab.vue中的<router-view>中的内容，类似这样只需要跳转页面，不需要添加验证方法的情况，可以使用<router-link>来实现导航
    <router-link to="/setting"> setting</router-link>

  • 页面跳转 A-B
    (1)方法一：

    <div class="box" @click="goB">点我跳转</div>
methods：{
    goB(){
        //1.字符串 this.$router.push('/B')
        //2.对象 this.$router.push(path:'/B')
        //3.命名的路由
        this.$router.push({ name:'B',params:{msg: this.msg}})
    }
}

    (2)方法二：
    采用require的方式

    Vue.use(Router)
    const A = r =>require.ensure([],()=>r(require(‘@/view/RouterTest/A.vue’)),’A’)
    const B = r =>require.ensure([],()=>r(require(‘@/view/RouterTest/B.vue’)),’B’)

    export default new Router({

    routes:[
    {path:'/a',name:'A',component:A},
    {path:'/b',name:'B',component:B},
]

    })

  • vue路由传参params与query两种方式的区别
    query更加类似于ajax中的get传参，params类似于post，前者在浏览器地址中显示参数，后者不显示。

  1. Axios
    安装 npm install axios –save
    引入 import axios from ‘axios’

  • get请求
    向具有指定id的用户发出请求 

    axios.get('/user?ID=123').then(function(response){
console.log(response)
}).catch(function(error){
console.log(error)
})

    通过params对象传递参数

    axios.get('/user',{
    params:{id:123}
}).then....

  • post请求 

    axios.post('/user',{
    firstName:'fred',
    lastName:'Flint'
}).then....
  1. vuex(组件的数据共享)
    利用vuex进行状态管理，把所有的事件和状态存储在store对象中，在组件中通过计算属性获得事件。不能直接改变store中的状态，唯一途径是显示地提交mutations.
  • (1)npm install vuex -save
    (2)新建一个store文件夹，新建store.js文件，引入vue和vuex
    (3)引用:Vue.use(Vuex)
    (4)在main.js中，引入store.js文件,实例化Vue对象时加入store对象
  • state
    单一状态树，保存项目状态唯一数据源，通过this.$store.state直接访问，mapState辅助函数
  • Getter
    定义getter方法，暴露出store.getters对象供访问，mapGetter辅助函数
    • Mutation
      更改store数据，定义mutations方法，通过 this.$store.commit()调用，mutation必须是同步函数
    • action
      可以包含异步操作，通过this.$store.dispatch()调用，结合Promise使用，实现组合action
  1. 命名规范
  • 文件和文件夹:小写字母中划线
  • 组件name配置:小写字母中划线
  • 组件及js变量和方法:首字母小写的驼峰
  • 组件引用:首字母大写的驼峰
  • .env环境配置遍历: 大写加划线
